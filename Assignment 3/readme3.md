# Advanced E-Commerce Web App

## Description
This is a dynamic fashion store web application built using HTML, CSS, JavaScript, and Bootstrap. It simulates an online shopping experience.

## Features
- Dynamic product listing
- Add to cart functionality
- Remove items from cart
- Cart sidebar UI
- Total price calculation
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- Bootstrap

## Future Enhancements
- Payment integration
- User login system
- Product filtering
