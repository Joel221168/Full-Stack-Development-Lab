# Travel Agency Website

## Description

This project is a simple Travel Agency web application built using **Node.js**, **Express.js**, and **MongoDB**.
Users can view travel destinations and book trips through a form. The booking information is stored in a MongoDB database.

## Technologies Used

* Node.js
* Express.js
* MongoDB
* Mongoose
* EJS
* HTML & CSS

## Features

* View travel destinations
* Book a travel package
* Store booking data in MongoDB

## How to Run

1. Install dependencies

```
npm install
```

2. Start the server

```
node server.js
```

3. Open in browser

```
http://localhost:3000
```
